## Quick start

### Install node

Recommended: install node via [NVM](https://github.com/nvm-sh/nvm)

### Install gulp

Install the gulp command line utility
```
> npm install --global gulp-cli
```

### Setup project

```
> npm install
```

```
> gulp build
```

```
> gulp watch
```


Open index.html and have fun!
